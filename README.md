# 月預算分配計算機

把收入分成四大類，幫助規劃每月預算的小工具。

## 四大分類

- 🌿 **固定支出**（建議 50%）：房租、水電、餐費、交通等每月必要開銷
- 💚 **年度攤提**（建議 30%）：保險、稅金、紅包、備用金等年度支出 ÷ 12
- 🌸 **自我經營**（建議 10%）：興趣、娛樂、成長學習
- 💜 **圓夢計畫**（建議 10%）：投資、緊急預備金、旅遊

比例僅供參考，可依個人狀況調整。

## 本機執行

需要先安裝 [Node.js](https://nodejs.org/)（18 以上版本）

```bash
npm install
npm run dev
```

開啟 http://localhost:3000 即可使用。

## 部署到網路上

推薦使用 [Vercel](https://vercel.com)（免費）：

1. 把專案上傳到 GitHub
2. 到 Vercel 選擇該 repo 連結
3. 等 1–2 分鐘自動部署完成即可

## 資料儲存

所有填入的資料都儲存在你自己瀏覽器的 localStorage，不會上傳到任何伺服器。
換裝置或清除瀏覽器資料後會重置。
